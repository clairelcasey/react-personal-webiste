Jobly React/ Express Application

Key components include:

  • Express/ Node.js back-end
  • Authorization/ authentication handled using JWTs and middleware
  • Test driven devlopment on back-end
  • Back-end validation using JSON Schema
  • React front-end including custom hooks (useLocalStorage)

Back-end Login/ Logout functionality handled through:

  • JSON Web Tokens (JWTs)
  • Authorization/ Authentication middleware
  • User class (OOP) and PostgreSQL database through Node-Postgres

Logged in users can:

  • Search for companies and jobs (live search)
  • View jobs associated with each company
  • Apply for a job

Key aspects of back-end include:

  • Test Driven Devlopment (TDD) - 100% coverage
  • User, Company, and Job classes and Node-Postgres queries
  • JSON Schema validation

Key aspects of front-end include:

  • React function components
  • React hooks including:
    • useState, useContext, useHistory, useCallback, and useLocalStorage (custom hook)

Key Takeaways:

  • Benefits and drawbacks of Test Driven Development for RESTful APIs
  • Importance of strategically thinking through component hierarchy for React Applications
