Historical Weather Interactive Visualization

Key components include:

  • Data from the World Weather API.
  • Modified WeatherWheel class created by Eric Lo.
  • Download script and data formatting

Hover over WeatherWheel to view:

  • Yearly and monthly statistics for weather conditions (e.g. sunny days)
  • Daily statistics including precipitation, high and low temperatures

Update data to show in WeatherWheel by either:

  • Selecting city by images at the top
  • Selecting year using dropdown selector

Codebase includes:

  • _downloads.js - back-end download script to download and format data based on a list of years and cities
  • WeatherWheel.js - external class modified for specific data and display
  • display.js - front-end script

Key Takeaways:

  • Benefits of using OOP when sharing code with others
  • Learning to modify components of code written by others
  • Learning to manipulate data from a third party API
