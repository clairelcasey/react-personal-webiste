BART Estimated Restoration Times

Key components include:

  • Data from the BART API.
  • Visualizations from Javascript D3 library
  • Bootstrap CSS

Use toolbar to change displays. Display options include:

  • Display delays
  • Display specific BART line
  • Display by Wait Time

Display by wait time to visualize where trains are along route.

Codebase includes:

  • main.js - create d3 map and update visualizations, listen for events d3 visualizations
  • filter.js - filter data given a specific BART line and/or direction
  • requests.js - make Axios AJAX requests to BART API
