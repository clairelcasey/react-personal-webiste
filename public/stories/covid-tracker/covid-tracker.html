New York Covid Tracker Application

Key components include:

  • Login/ logout functionality
  • Visualizations from Javascript D3 library
  • Object oriented programming
  • Flask back-end
  • Data from the New York State Department API.

Login to add a tracker.

Login/ logout authorization handled using:

  • WTForm validation
  • Flask's Session

Click to remove a followed county.

Dynamic bar charts for each county a user is following:

  • Dynamic tooltip
  • Flask's Session

Uses SQLAlchemy as ORM & Object Oriented Programming
