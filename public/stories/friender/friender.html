Friender React/ Flask Application

Key components include:

  • Flask back-end
  • Authorization/ authentication handled using JWTs
  • Geographic data analysis using Mapbox API and GeoPy
  • Image storage using AWS S3
  • React front-end

Back-end Login/ Logout functionality handled through:

  • JSON Web Tokens (JWTs)
  • Local Storage
  • User class (OOP) and PostgreSQL database through SQLAlchemy

Logged in users can:

  • Scroll through potential friends
  • Like or dislike potential friends

Users can update their profile image (images stored in AWS S3)

Potential friend matches calculated using User class:

  • Liking/ disliking methods and database relationships
  • Lat/Long and distance calculations using Mapbox API and GeoPy library

Key Takeaways:

  • Importance of strategically thinking through pros/ cons of front-end and back-end technologies and frameworks
  • Time estimation for full-stack development projects
